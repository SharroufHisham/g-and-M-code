## Operations Mapping (from Fiche de Réglage + Programme)

### OP10 (Main spindle — right side, bar stock)
- Face and rough external profile (Ø27, hex=24 area, tapers)
- Center drill Ø16 (90°)
- Drill Ø9.8
- Drill Ø6 (deep hole)
- Drill Ø9.5 (HSS 180°, ream step)
- Finish external profile (shoulders, tapers, thread relief)
- Thread external (OP10 program uses F1.5 — ⚠️ this is NOT the G1/2 BSP thread; the G1/2 BSP at F1.814 is cut in OP20)
- Finish thread pass
- Ream/finish internal bore (Ø10 fitted, Ø6.944 step per program)
- Cut off at overall length ~34

### OP20 (Sub-spindle — left side)
- Face and rough external profile
- Drill
- Rough internal bore
- Finish external profile
- Thread G1/2 external from OP20 side (pitch 1.814 mm, BSP 14 TPI)
- Finish internal bore per profile R306 251 00
